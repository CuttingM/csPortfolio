Lightning

One of the challenges I had during this project was determining how each segment of the lightning bolt’s length, angle, and strokeWeight would be determined. I decided it would be interesting to leave this up to the caller of the function that draws the bolt. The function can be supplied three other functions as arguments to control each of the three properties (length, angle, and strokeWeight). This was easy to use and experiment with in p5 by using lambda expressions, but it required some more work to convert to Java because the current version of Processing doesn’t support lambdas.
